## What This Release Is

FerrumGate `v0.1.0-rc1` is a self-hosted execution-governance gateway for MCP/agentic operations. It provides:

- Policy-evaluated, capability-bounded, approval-aware, rollback-classified, provenance-tracked execution lifecycle
- Bearer token authentication with scoped token/RBAC model (Phase 4 complete)
- SQLite (single-node) and PostgreSQL (single-node primary/standby) storage backends
- Local SLO validation baseline and sustained observation window infrastructure
- Read-only operator TUI convenience (`ferrum-tui`)
- Helm/K8s packaging scaffold (DEP-5 KIND smoke passed)
- MCP target-mode smoke (15/15 checks passed 2026-05-21)

This release is appropriate for **operator-accepted domainless pilot evaluation** on single-node SQLite or PostgreSQL topologies. It is **NOT** appropriate for production workloads, multi-tenant deployments, or unattended automated failover.
