# 05 — Done criteria

Một phase chỉ xong khi:
- code ở scope phase đó hoàn tất
- docs cập nhật
- contracts/schemas/openapi không drift
- tests hoặc smoke coverage cập nhật
- invariants vẫn giữ

Not done nếu:
- chỉ thêm TODO
- chỉ compile mà không reflect docs/spec
- thêm mutation path nhưng không có rollback/provenance
