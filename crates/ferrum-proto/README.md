# ferrum-proto

Source-of-truth Rust types for FerrumGate.

## Includes

- Strong identifiers
- Common enums
- Intent types
- Capability types
- Provenance types
- Rollback types
- API request/response types

## Principles

- Types must map to `schemas/` and `openapi/` definitions.
- Any shape change must be reflected in documentation and contracts.
